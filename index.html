Transcriptomics of Bronchoalveolar Lavage Cells Identifies

New Molecular Endotypes of Sarcoidosis

View On GitHub

    This is the supplementary website for the manuscript titled "Transcriptomics of Bronchoalveolar Lavage Cells Identifies New Molecular Endotypes of Sarcoidosis", where you can download the data, the supervised analysis results, the unsupervised analysis results and the R codes we used to generate the figures in the manuscript.

    You can also download all the files by clicking "View On GitHub" under the title, which will lead you to the github repo download.

    Supervised Analysis

    Supplementary files for the supervised analysis include:

    • Lists of genes associated with clinical traits [.zip]
    • Pathway enrichment analysis of the genes associated with clinical traits [.zip]

    Unsupervised Analysis

    • Assignment of genes to the 48 identified gene modules [.xls]
    • Pathway enrichment analysis results for each module [.zip]
